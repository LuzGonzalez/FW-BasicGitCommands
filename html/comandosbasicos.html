COMANDOS BÁSICOS DE GIT.

Este comando crea un nuevo repositorio de git vacío, o reinicializa uno ya existente.

Ejemplo de uso:
git init 
            Ir a comandos para crear ramas.

Es usado para establecer una configuración especifica de usuario.

Ejemplo de uso:
git config --global user.name "user" 
                Ir a comandos para crear ramas.

Este comando se usa para agregar archivos al area de stagging.

Ejemplo de uso:
git add archivo.extension

Este comando agrega un archivo en el HEAD, pero aún no en tu repositorio remoto.

Ejemplo de uso:
git commit -m "message"

Este comando clona un repositorio en un directorio recien creado.
